Suma y resta de vectores

Ejercicios interactivos para practicar componentes, suma, resta, módulo y dirección.

Recordemos

Un vector puede escribirse con sus componentes: A = (Ax ; Ay). Para sumar o restar vectores, se trabaja componente por componente.

A + B = (Ax + Bx ; Ay + By)
A - B = (Ax - Bx ; Ay - By)
|R| = √(Rx² + Ry²)

En los ejercicios con módulo, escribí el resultado con 2 decimales si hace falta.
